Docker Desktop is required to build and run
your QuickCart application in containers.

==============================================
  STEP 1: Install Docker Desktop
==============================================

1. Download from:
   https://www.docker.com/products/docker-desktop

2. Install Docker Desktop for Windows

3. Restart your computer

4. Start Docker Desktop application
   (Find in Start Menu)

5. Wait until Docker icon in system tray
   stops animating (engine is ready)

==============================================
  STEP 2: Verify Docker is Working
==============================================

Open PowerShell and run:

   docker --version

You should see: Docker version 24.0.x
(or similar)

==============================================
  STEP 3: Build QuickCart Images
==============================================

Once Docker is installed and running:

Option 1: EASIEST
-----------------
Double-click: docker-rebuild.bat


Option 2: PowerShell
--------------------
cd C:\Users\User\Desktop\QuickCart
docker compose up --build -d
